# 🛒 E-Commerce Web Application (React)

A modern **E-commerce web application** built using **React.js** and **Vite**, focused on clean UI, component-based architecture, and smooth user experience.

Live Url : https://react-ecommerce-azure-zeta.vercel.app

---

## 🚀 Features
- Product listing with dynamic rendering
- Reusable React components
- Fast development setup using Vite
- Clean and responsive UI
- Organized project structure

---

## 🛠 Tech Stack
- **Frontend:** React.js, JavaScript, HTML, CSS
- **Build Tool:** Vite
- **Package Manager:** npm
- **Version Control:** Git & GitHub

## ⚙️ Installation & Setup

1. **Clone the repository**
```bash
git clone https://github.com/Shayan464/E-commerce-Project-using-react-js.git
````

2. **Navigate to the project folder**

```bash
cd E-commerce-Project-using-react-js
```

3. **Install dependencies**

```bash
npm install
```

4. **Run the development server**

```bash
npm run dev
```

The app will run on:

```
http://localhost:5173
```
